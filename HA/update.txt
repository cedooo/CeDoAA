UPGRADE
Stop Home Assistant

Open the directory where the virtual environment is located:
$ cd homeassistant
Activate the virtual environment:
$ source bin/activate
Upgrade Home Assistant:
$ python3 -m pip install --upgrade homeassistant
Start Home Assistant
